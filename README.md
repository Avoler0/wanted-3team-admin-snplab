## 팀원 및 구현 기능
### 조혜빈
- 메인페이지 내에서 지원자 리스트 컴포넌트(ApplicantsList) 구현
- 받은 데이터에서 회차별 필터링
- 데이터에 회차 개수 만큼 탭 개수 구현
- 회차별 필터링한 데이터 내에서 페이지네이션 구현
- 체크박스 클릭시 지원자 당첨여부 업데이트

### 이성진
- 앱화면(크라우드 워커 지원 페이지)
  - 정보 입력
    - form 입력, 검증 구현
    - 메뉴 표시를 위한 모달 구현
    - 상호작용에 의한 CSS 적용
  - 거주지역 선택
  - 개인정보 처리방침
  - 제3자 정보제공 동의 안내
  - 입력완료
  - 지원 완료
